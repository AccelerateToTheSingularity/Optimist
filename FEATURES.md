# Optimist Prime — What This Bot Does for r/accelerate

AI-powered community assistant for [r/accelerate](https://www.reddit.com/r/accelerate/), running 24/7 on GitHub Actions. Open source, feature-toggled, and built to help the community stay informed while giving moderators smarter tools.

## For everyone

- **Smart TLDRs** — Long posts and comments summarized automatically so you can catch up fast
- **Thread digests** — Discussion summaries as conversations hit milestones (20, 50, 100 comments)
- **Linked-post summaries** — Crossposts and Reddit links explained in plain English when a post is mostly a pointer elsewhere
- **Summon the bot** — Address Optimist Prime directly (“hey bot”, “Optimist Prime”, etc.) for help or context
- **Natural conversations** — The bot replies when you engage with it in-thread

## For the community

- **Acceleration flair** — Opt-in reputation tiers reflecting your focus across pro-AI communities
- **Contributor recognition** — Milestone and specialist flairs celebrating active members
- **Curated crossposts** — Top AI-related posts from r/accelerate shared to [r/ProAI](https://www.reddit.com/r/ProAI/)

## For moderators

- **Discrete AI moderation** — Many independent rules, each evaluated precisely (not one vague mega-prompt)
- **Mod alerts with context** — AI-written summaries in reports and modmail so you know what to verify
- **Troll early-warning** — Flags accounts with persistently downvoted local comments for human review
- **Safe mode & audit trail** — Dry-run everything before live actions; actions logged for accountability
- **Leading AI models** — Powered by industry-standard backends including OpenAI and Gemini

## Built responsibly

- Open source on GitHub
- Feature toggles for every major capability
- Rate limits and daily caps on AI usage
- Duplicate-action prevention so the bot does not spam
- Destructive actions (bans, removals) behind explicit toggles — safe defaults first
